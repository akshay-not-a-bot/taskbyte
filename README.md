
# TaskByte 📝
TaskByte is a simple yet functional to-do list web application built to showcase my skills in Django, Bootstrap, and PostgreSQL, along with cloud deployment using Koyeb. This project serves as a demonstration of my ability to develop full-stack web applications using Django’s powerful backend capabilities, a PostgreSQL database, and a responsive Bootstrap-based UI.

## Key Features
### 🚀 Django-Powered Backend
- Implements Django’s MTV (Model-Template-View) architecture for clean and scalable development.
- Uses Django ORM to interact with the PostgreSQL database efficiently.
- Implements user authentication (login/logout, registration) with Django’s built-in authentication system.

### 🅱️ Bootstrap-Enhanced UI
- Responsive and mobile-friendly design using Bootstrap 5.
- Clean and minimalistic task management interface.

### 🗄 PostgreSQL Database
- Uses PostgreSQL for robust and scalable data storage.
- Efficient querying and data management through Django’s ORM.

### ☁ Cloud Deployment on Koyeb
- Deployed on Koyeb, leveraging its free-tier cloud hosting for smooth and continuous deployment.
- Integrated with GitHub Actions for automated deployment.

### Why This Project?
This project was created to demonstrate my proficiency in Django development, from setting up a robust backend to integrating a PostgreSQL database, designing a responsive UI with Bootstrap, and deploying it to the cloud. It highlights my ability to work with full-stack web technologies and deploy real-world applications efficiently.
